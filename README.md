# ⚽ Creador de grupos

## Descripción

**Creador de grupos** es una aplicación web desarrollada en HTML, JavaScript y TailwindCSS que permite simular una fase de grupos de fútbol con 4 equipos. El sistema calcula automáticamente la tabla de posiciones en base a los resultados ingresados por el usuario, incluyendo goles y estadísticas disciplinarias.

El objetivo del proyecto es ofrecer una herramienta interactiva, visualmente moderna y funcional para la simulación de competencias deportivas.

---

## Características principales

- Configuración editable de los nombres de los 4 equipos.
- Generación automática de los 6 partidos de la fase de grupos (todos contra todos).
- Ingreso manual de resultados por partido:
  - Goles a favor de cada equipo
  - Tarjetas amarillas 🟨
  - Tarjetas rojas 🟥
- Cálculo automático de la tabla de posiciones.
- Sistema de puntuación estándar:
  - Victoria: 3 puntos
  - Empate: 1 punto
  - Derrota: 0 puntos
- Ordenamiento por:
  1. Puntos
  2. Diferencia de gol
  3. Goles a favor
- Clasificación automática de los 2 primeros equipos.
- Indicador visual de:
  - 🟢 Clasificado
  - 🔴 Eliminado
- Interfaz moderna con TailwindCSS y animaciones suaves.

---

## Tecnologías utilizadas

- HTML5

- TailwindCSS (CDN)

No requiere instalación ni dependencias adicionales.
